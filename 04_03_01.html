WEB prototype

Link til site

I dette project skulle vi arbejde med Adobe XD hvor vi skulle lave en prototype med ny Fronter-løsning. Dette skulle være en mobile first/only web prototype som vi efterfølgende skulle kode i hhv. HTML, CSS, JAVA script. Prototypen skulle indeholde mindst en formular med forskellige input typer samt have mindst 1 funktionelt user flows med task og subtasks. Prototypen skulle tage udgangspunkt i KEAs visuelle stil.

Til vores ide udvikling brugte vi The Four-Step sketch, hvor vi først skrev alle vores tanker ned på et papir individuelt. Dernæst tegnede vi skitser af vores ideer på et nyt papir. Det næste skridt var Crazy 8s hvor vi lavede 8 forskellige tegninger med 1 minut til hver. Til sidst lavede vi solution sketch hvor vi arbejdede mere detaljerede på de ideer, fra vores individuelle papir, vi fandt bedst. Her skulle tegningen kunne forklare sig selv og vi hang dem op på væggen bag ved os og da alle havde gjort det snakkede vi sammen i gruppen om hvilke ideer vi godt kunne tænke os at arbejde videre med.
